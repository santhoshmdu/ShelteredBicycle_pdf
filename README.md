## Project Overview

An accessible, weather-protected electric vehicle prototype designed for general and differently-abled users, focusing on sustainable urban mobility.

## Key Features

- **Inclusive Design**: Accessible frame for general and differently-abled users
- **Weather Protection**: Enclosed design for all-weather commuting
- **Smart Integration**: IoT sensors and embedded systems
- **Ergonomic Focus**: User-centric design for enhanced comfort
- **Sustainable Materials**: Eco-friendly material selection

## Technologies Used

- **CAD Design**: Siemens NX CAD
- **Electronics**: Embedded Systems, Arduino/ESP32
- **Sensors**: IoT sensors for monitoring and control
- **Mechanical**: Structural analysis and optimization

## Design Highlights

### Frame Design
- Weather-protected enclosure
- Accessibility features for differently-abled users
- Lightweight yet durable construction
- Ergonomic seating and controls

### Smart Features
- Real-time battery monitoring
- GPS tracking
- Speed and distance sensors
- Mobile app connectivity

### Sustainability
- Recyclable materials research
- Energy-efficient motor system
- Minimal environmental footprint
- Long-lasting components

## Project Workflow

1. **Research Phase**: User needs analysis, material research
2. **Design Phase**: NX CAD modeling, ergonomic optimization
3. **Prototyping**: Physical prototype development
4. **Integration**: Embedded systems and IoT implementation
5. **Testing**: User trials and performance evaluation
